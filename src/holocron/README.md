# Holocron - Deprecated

This system is no longer actively used. Modern agentic IDEs provide superior semantic search capabilities.

## Historical Context
Holocron was a vector database system for semantic documentation search. It has been superseded by:
- IDE-native semantic search
- Codebase-wide indexing in modern AI tools
- Direct access to external_docs/ and OpenAPI specifications

## Preserved Features
The code remains for historical reference and potential future use cases:
- Vector database management patterns
- Semantic chunking strategies
- Documentation processing workflows

## Migration Path
For semantic search capabilities, use:
- **IDE Semantic Search**: Built-in codebase indexing in modern IDEs
- **External Documentation**: Direct access to `external_docs/` directory
- **OpenAPI Specification**: Reference `external_docs/openapi-swagger.json`
- **User Documentation**: Browse `external_docs/user-docs/` directory

## Deprecation Notice
- **Status**: Deprecated as of 2025
- **Reason**: Superseded by modern IDE capabilities
- **Action**: Use IDE-native search instead of holocron commands
- **Maintenance**: Code preserved for reference only
