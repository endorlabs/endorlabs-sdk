# Rego Policies (SDK)

The SDK exposes policy resources via `endor_cockpit.resources.policy`: list, get, create, update, delete. Policy content (Rego) is carried in the payload; create/update send the policy definition to the API. For Rego language reference, policy types, resource kinds, and field mappings see official Endor Labs Rego/policy documentation (e.g. docs.endorlabs.com or external_docs). See notebooks for policy examples.
