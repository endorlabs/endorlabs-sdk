# AI access

Learn which features in the application use AI and how to manage their access.

The following features in the Endor Labs application access Artificial Intelligence (AI) services to enhance security analysis, code insights, and developer assistance. You can check whether AI access is currently enabled or disabled for these features.

* **[LLM code processing](../ai-llm/)**: Detects AI models from HuggingFace used in Python projects and lists them as dependencies.
* **[DroidGPT](../droidgpt/)**: Retrieves data from third-party AI tools and correlates it with Endor Labs’ proprietary risk data to identify open source software packages.
* **[C/C++ embeddings](../../scan-with-endorlabs/language-scanning/c/)**: Enhances detection capabilities of C and C++ software composition analysis (SCA) by using AI-generated embeddings.
* **[Endor AI Chat](../../ai/ai-chat/)**: Enables users to ask natural language questions about security findings, scan results, package versions, and vulnerabilities directly within the Endor Labs application.
* **[AI Security Review](../../ai/ai-security-review/)**: Enables users to identify potential security issues in pull requests and get recommendations.

### Modify AI access

To modify AI feature settings:

1. Select **Settings** > **AI Access** from the left sidebar.
2. Click **Contact us** to submit a request to the support team.
3. The support team will assist you with enabling or disabling AI features based on your organization’s needs.
