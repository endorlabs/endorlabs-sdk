# Quick start

Get up and running quickly with Endor Labs.

Log in to [Endor Labs](https://app.endorlabs.com) and select **Getting Started** from the left sidebar to view the options to get started.

You can choose from the following options:

* [endorctl](../quickstart/quickstart-local-system/)
* [GitHub App](../quickstart/quickstart-github-app/)

You can also use GitHub Actions to scan demo projects.
