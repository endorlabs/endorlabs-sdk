# Commands

Learn more about the available endorctl commands

---

##### [artifact sign](/endorctl/commands/sign-artifacts/)

Use the `artifact sign` command to sign container images and build artifacts in the CI pipeline.

##### [completion](/endorctl/commands/completion/)

Use the completion command to get a command completion script for a specified command shell

##### [container](/endorctl/commands/container/)

Use the container command to scan container images.

##### [endorctl api](/endorctl/commands/api/)

Use the endorctl api command to interact with the Endor Labs API

##### [help](/endorctl/commands/help/)

Use the help command to get command help for endorctl.

##### [host-check](/endorctl/commands/host-check/)

Use the host check command to verify if your system is appropriately setup to perform a scan.

##### [init](/endorctl/commands/init/)

Use the init command to authenticate to Endor Labs from a workstation with an external identity provider.

##### [sbom](/endorctl/commands/sbom/)

Use the sbom command to import or export SBOMs to or from Endor Labs

##### [scan](/endorctl/commands/scan/)

Use the scan command to perform endorctl scan.

##### [sync-org](/endorctl/commands/sync-org/)

Use the sync-org command to sync all projects in a GitHub organization to Endor Labs.

##### [toolchains](/endorctl/commands/toolchains/)

Use the toolchains command to detect the tools in your repository and create a scan profile.

##### [validate policy](/endorctl/commands/validate/)

Use this command to validate policies

##### [recommend](/endorctl/commands/recommend/)

Use the recommend command to suggest dependency updates that address issues across your environment.
