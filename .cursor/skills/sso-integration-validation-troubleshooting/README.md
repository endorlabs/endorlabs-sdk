# SSO Integration Skill Assets

This directory contains customer-facing workflow assets for Enterprise SSO setup and troubleshooting.

## Files

- `SKILL.md` - Intake-first workflow and troubleshooting guidance.
- `sso_access_spotcheck.py` - SDK-based evidence collector for claim-to-namespace mapping.

## Principles

- Use platform behavior language, not backend implementation references.
- Separate authentication success from authorization scope.
- Treat namespace targeting and propagation as independent policy dimensions.
